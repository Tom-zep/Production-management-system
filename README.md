# 棚卸調査票 自動作成ツール

棚卸作業で毎回手作業で行っていた①〜㉒の手順を自動化するPythonスクリプトです。  
TxZAIK / TxPRTS / TxTANK / TxITEM / GENKA_SUB の各CSVから棚卸調査票Excelを自動生成します。

---

## ファイル構成

```
棚卸自動化/
├── README.md                    # このファイル
├── 棚卸し調査票自動作成.py       # メインスクリプト
└── excel_diff.py                # 差分確認ツール
```

---

## 必要な環境

- Python 3.8 以上
- 以下のライブラリ（初回のみインストール）

```
pip install pandas openpyxl numpy
```

---

## 使い方

### 1. CSVファイルを準備する

以下のCSVファイルをスクリプトと **同じフォルダ** に置いてください。

| ファイル名 | 内容 | 必須 |
|---|---|---|
| `TxZAIK.CSV` | 在庫データ | ✅ 必須 |
| `TxPRTS.CSV` | 部品構成データ | 任意 |
| `TxTANK.CSV` | 単価データ | 任意 |
| `TxITEM.CSV` | 品目マスタ | 任意 |
| `GENKA_SUB_*.CSV` | 製品明細CSV（複数可） | 任意 |

### 2. スクリプトを実行する

PowerShell（またはコマンドプロンプト）で以下を実行：

```
python 棚卸し調査票自動作成.py
```

### 3. 出力ファイルを確認する

同じフォルダに以下のファイルが生成されます：

```
棚卸調査票自動作成.xlsx
```

---

## 設定変更（必要な場合）

スクリプト冒頭の設定部分を編集してください：

```python
INPUT_DIR   = "."                                   # CSVがあるフォルダ（デフォルト: 同じフォルダ）
OUTPUT_FILE = "棚卸調査票自動作成.xlsx"   # 出力ファイル名
TANKO_DATE  = ""                                    # 棚卸日（例: "20250630"）空欄なら自動検出
```

---

## 自動化している処理

| ステップ | 内容 |
|---|---|
| Step 1 | CSVファイルを読み込む（Shift-JIS / UTF-8 自動判定） |
| Step 2 | TxZAIK 整理・ソート |
| Step 3 | 棚卸日を自動検出 |
| Step 4 | GENKA_SUB 結合・積上原価計算（積上原価 ÷ 展開使用量） |
| Step 5 | TxPRTS / GENKA_SUB / TxITEM / TxTANK を VLOOKUP 結合 |
| Step 6 | 新積上原価を計算 |
| Step 7 | HOKAN → CODE 昇順ソート |
| Step 8 | Excel へ出力（書式・色付き） |

## 差分確認ツールの使い方

前回の正解ファイルと自動作成ファイルを比較して差分を確認できます。

```
python excel_diff.py 正解ファイル.xlsx 自動作成ファイル.xlsx
```

出力例：

```
■ ヘッダー行（行5）比較
  列11: '新積上原価'  ✓
  列12: 'GENKA'      ✓

■ データ差分（先頭50件まで）
  行6 列11[K]: 正解=246.40  →  自動作成=246.40

■ 列ごとの差分件数（多い順）
  新積上原価: 0件
  購買単価:   0件
```

---

## 注意事項

- CSVファイルはダブルクリックで直接開かず、スクリプトに読み込ませてください
- 文字コードは Shift-JIS / CP932 / UTF-8 を自動判定します
- `GENKA_SUB_*.CSV` は複数ファイルあっても自動で結合します
- 比較ツールを使う際は、正解版の `.xlsm` を一度 `.xlsx` で保存してから使用してください
